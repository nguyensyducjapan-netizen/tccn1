# FinFlow - Personal Finance App

A YNAB-like personal finance application built with React, Tailwind CSS, and Supabase.

## Features
- **Zero-based Budgeting:** Give every dollar a job.
- **Expense Tracking:** Log transactions and categorize them.
- **Reports:** Visualize your spending (Coming Soon).

## Setup

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Environment Setup**
   Ensure `.env` contains your Supabase credentials:
   ```
   VITE_SUPABASE_URL=...
   VITE_SUPABASE_ANON_KEY=...
   ```

3. **Run Locally**
   ```bash
   npm run dev
   ```

## Tech Stack
- **Frontend:** React, Vite, TypeScript, Tailwind CSS
- **Backend:** Supabase (Auth, Postgres)
- **Icons:** Lucide React
